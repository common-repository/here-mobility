== Description ==

The HERE Mobility Web Widget is a customizable transportation booking widget, that you can quickly add to your website, to offer your users ride booking and navigation.
The Web Widget supports pickup location, destination, and required departure time, in addition, your users can also customize their ride according to any special requirements they have from suitcases to number of passengers.
The HERE Mobility Web Widget connects to the Mobility Marketplace to find the most relevant rides matching your users’ requests and displays them to your users to choose from and book.

To activate your Web Widget, please create an [account](https://developer.mobility.here.com/signup) and request your app credentials.

= The following languages are currently supported in the widget: =
* Dutch
* Finnish
* English
* French
* Spanish
* Greek
* Portuguese

= Special Features =

* Seamless Mobility Checkout: You can pass parameters such as name, phone number, pickup date/time, and number of passengers from your checkout to the Web Widget so your user doesn’t have to enter them twice.
* Real Time Tracking: Once your user has booked a ride they will receive real time ride updates from the Web Widget ride tracker, specifying the current ride status, location and ETA.
* Brand Customization: You can customize the HERE Mobility Web Widget to your brand and website look and feel (font, etc).
* Analytics: You can retrieve analytic reports to examine your user’s behaviors and use of mobility services in conjunction to your services.
* Mobile Responsive: The HERE Mobility Web Widget is mobile responsive, adapting its display to the device it’s running on.

== Installation ==

1. Navigate to the plugin section in the left menu
2. Look for here mobility plugin and install it
![Here Mobility Web Widget plugin in marketplace](https://ps.w.org/here-mobility/assets/screenshot-1.png "Here Mobility Web Widget plugin in marketplace")
3. Once the plugin is installed, go to the plugins settings
![Here Mobility Web Widget plugin Settings page](https://ps.w.org/here-mobility/assets/screenshot-2.png "Here Mobility Web Widget plugin Settings page")
4. Go to the Here Mobility Settings page and fill in the App Key and App Secret values. (See Frequently Asked Questions below to learn how to get these values.)
Here you will be able to set:
* Your app keys
* You app secret
* Whether you want to use a random user ID or your own user ID
* Set default pick up or destination (optional)
5. Select the reGo to Post's editor and insert the widget by clicking on the following button
![Add button in WYSIWYG editor](https://ps.w.org/here-mobility/assets/screenshot-3.png "Add button in WYSIWYG editor")
in the the page where you want to display Here Mobility web widget
6. Once you click this button, you will have the option whether to use the default settings you previously set or to set new settings that will apply only for this specific page
![Customize default settings](https://ps.w.org/here-mobility/assets/screenshot-4.png "Customize default settings")
7. Publish your changes.

== FAQ ==

How can I get my app key and secret values?

In order to use the HERE Mobility Web Widget, you'll need an application ID, an application key.
To obtain the authentication credentials, please create an [account](https://developer.mobility.here.com/signup) and request your app credentials.
